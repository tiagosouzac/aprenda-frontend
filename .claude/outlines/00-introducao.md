# Outline — Módulo 00: Introdução ao curso

**Fio condutor**: Antes de entrar no primeiro conteúdo técnico, o estudante precisa entender o que vai aprender, o que vai construir, e como o curso está organizado. Este módulo tem uma única lição — serve como orientação inicial, não como conteúdo técnico.

**Tom**: direto e informativo. Sem hype. O estudante chegou aqui para aprender — este módulo mostra o mapa.

---

## Lição 1: Bem-vindo ao curso

### Seções (H2)

#### O projeto do curso
- O fio condutor é o blog `artigo.dev` — construído e reconstruído em cada módulo com uma nova camada
- HTML: documento. CSS: estilo. JavaScript: interatividade. TypeScript: tipos. React: componentes. Testes: confiança.
- Cada versão é independente — mas o estudante vê onde cada tecnologia resolve o problema que a anterior não resolvia

#### Estrutura do curso
- 8 módulos em sequência: Web, HTML, CSS, JavaScript, TypeScript, React, Testes, Próximos passos
- Os 3 primeiros constroem modelo mental; os 4 seguintes são progressivamente práticos; o último é um mapa
- **Código**: diagrama ASCII dos 8 módulos com descrição de uma linha cada

#### Pré-requisitos
- Nenhum pré-requisito técnico — o curso começa do começo
- Familiaridade básica com o computador (terminal, pastas, instalar programas) ajuda a partir do módulo de JavaScript
- Quem já programou: os primeiros módulos serão rápidos. Quem nunca programou: vá no ritmo que funcionar

#### Como usar o curso
- Cada lição tem resumo ao final — útil para revisão
- Exemplos de código são para ser executados, não só lidos
- Quizzes: responder antes de ver a resposta — o objetivo é descobrir o que ainda não se entendeu
